# WPF 玩具合集
这个仓库是我用来存放和分享 Wpf 小玩具的地方，目的是学习Wpf

## ✨ 现有功能
### 🧮 计算器 (Calculator)
### 🎬 媒体播放器 (MediaPlayer)

## 📁 项目结构
├── src/ # 源代码目录
│ ├── Calculator/ # 计算器项目
│ │ ├── MainWindow.xaml # 计算器界面
│ │ └── MainWindow.xaml.cs # 计算器逻辑代码
│ └── MediaPlayer/ # 媒体播放器项目
│ ├── MainWindow.xaml # 播放器界面
│ └── MainWindow.xaml.cs # 播放器控制逻辑
├── WpfExample.slnx # 解决方案文件
└── README.md # 项目说明
